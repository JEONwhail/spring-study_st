### 4장. 서블릿과 스프링MVC

```
- mvc1 패턴
- mvc2 패턴
- MVC 패턴
```

### 실행
```
mvc1의 폼 실행: http://localhost:8080/view/v1/form.jsp
mvc1의 저장 실행: http://localhost:8080/view/v1/save.jsp

mvc2의 폼 실행: http://localhost:8080/servlet/form
mvc2의 저장 실행: http://localhost:8080/servlet/save

MVC의 예제 실행: http://localhost:8080/sample/ex1

```

### 에러

```
- 브라우저에서 요청시 jsp가 다운로드 될때:
  jstl 관련 라이브러리 추가 및 브라우저 캐시 삭제
  
- jsp에서 자바코드가 인식안될 때:
  표현식이 잘못되지 않았는지 확인 
```
